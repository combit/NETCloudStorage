# Project Description
To access the cloud storage services Dropbox, Google Drive and Microsoft OneDrive with List & Label, use the classes provided in the corresponding assemblies. Please note that they require additional files from their respective projects via NuGet.

# Further Information
Depending on the cloud service, you need to provide authentication information like Application-ID, Secret, Refresh-Token etc. Refer to the documentation of the corresponding NuGet package/cloud service provider for further information.
